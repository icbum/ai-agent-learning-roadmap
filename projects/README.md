# 项目模板总览

这个目录提供 4 个项目模板，对应这份学习路线图里的 4 类核心作品。

## 项目列表

1. [CLI Tool-Using Agent](01-cli-tool-using-agent.md)
2. [Research / RAG Agent](02-research-rag-agent.md)
3. [Coding / Workflow Agent](03-coding-workflow-agent.md)
4. [Production-Style Agent System](04-production-style-agent-system.md)

## 如何使用这些模板

每个模板都包含：
- 项目目标
- 核心能力
- 最小功能范围
- 推荐技术栈
- README 应该写什么
- 求职时如何讲这个项目

建议做法：
- 先做最小可运行版本
- 再补日志、配置、错误处理
- 最后补 README、架构图、评估说明

## 目标

这 4 个项目不是为了凑数量，而是为了覆盖 AI Agent 岗位最关键的能力面：
- tool calling
- retrieval / memory
- orchestration / workflow
- observability / eval / deployment
